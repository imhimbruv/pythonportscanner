# ⚡ Python Port Scanner

Fast multi-threaded port scanner with clean CLI and file export.

## Features
- Multi-threaded scanning
- Colored terminal output
- Service detection
- CLI arguments
- Export results to file

## Usage
```bash
python port_scanner.py -t scanme.nmap.org -p 1-1000
```

## Save results
```bash
python port_scanner.py -t scanme.nmap.org -p 1-1000 -o results.txt
```

## Disclaimer
Use only on systems you own or have permission to scan.
